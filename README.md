#  Lancement du Script d'Installation Postinstall

Ce script permet d'automatiser l'installation et la configuration initiale de votre système.

## Étapes d'Exécution

Pour télécharger et lancer le script, exécutez les commandes suivantes dans votre terminal, en tant que `root` ou avec `sudo` :

1.  **Télécharger le script :**
    ```bash
    curl -o postinstall.sh [https://raw.githubusercontent.com/razen377/Postinstall/refs/heads/main/postinstall.sh](https://raw.githubusercontent.com/razen377/Postinstall/refs/heads/main/postinstall.sh)
    ```

2.  **Rendre le script exécutable :**
    ```bash
    chmod +x postinstall.sh
    ```

3.  **Lancer le script :**
    ```bash
    sh postinstall.sh
    ```
